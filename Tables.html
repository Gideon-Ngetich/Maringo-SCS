When creating tables we are required to use validation rules in order to restrict data input in a table field. They can also be used in input forms. Validation rules help the user to input valid dataa that is required to give accurate results. Some of the common validation rules that we are giong to use are:

  • <>0 - Enter a nonzero value.
  • >=0 - Value must be zero or greater or You must enter a positive number.
  • Date() - Birth date cannot be in the future.
  • IS NOT NULL - The field should not be empty.
  • AND - Specifies that all parts of the validation rule must be true..
